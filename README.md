## üìñ Overview
**DeluxeMentions** is a professional-grade Minecraft plugin designed to enhance player communication. It provides a robust system for player mentions in chat, featuring highly customizable visual and auditory feedback to ensure important messages are never missed.

## ‚ú® Key Features
* **Smart Detection:** Real-time `@Player` mention detection in chat.
* **Immersive Notifications:** Send Titles, ActionBars, and BossBars to mentioned users.
* **Audio Feedback:** Customizable notification sounds for a better user experience.
* **Modern Formatting:** Full support for MiniMessage and Hex/RGB colors.
* **User Control:** Command for players to toggle notifications on or off according to their preference.

## üîß Technical Details
* **Language:** Java 21.
* **Platform:** Paper / Spigot.

## üöÄ Build from Source
To compile the project, ensure you have JDK 21 installed on your system:

```bash
git clone [https://github.com/RubenArtz/DeluxeMentions.git](https://github.com/RubenArtz/DeluxeMentions.git)
cd DeluxeMentions
./gradlew clean shadowJar

```

The compiled `.jar` file will be located in `build/libs/`.
